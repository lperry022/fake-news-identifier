About this project

Fake News Identifier helps people quickly assess the credibility of headlines and links. It highlights suspicious keywords, shows a source reputation badge, and provides an overall credibility score. Built by Group 11 for our Applied Software Engineering project.

Features
  • bolt Real-time analysis in the browser
  • badge Source reputation badges (Trusted / Untrusted / Unknown)
  • accessibility Accessible UI (keyboard & screen-reader friendly)
  • speed Lightweight — Materialize + Vanilla JS only
  • privacy_tip Privacy-respecting: no personal data required
Technology Stack (mandated)
Node.js (backend)
Vanilla JavaScript
HTML & CSS
Materialize CSS
MongoDB

Frontend calls /api/analyze and /api/health on the backend.
